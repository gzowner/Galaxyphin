# Galaxyphin independence

Galaxyphin used Wholphin as a one-time source import only.

From version 0.1.0 forward:

- all source changes are made in `gzowner/Galaxyphin`;
- Galaxyphin versions and tags belong to this repository;
- APK builds and releases are produced from this repository;
- no automatic upstream synchronization is configured;
- the Wholphin link is retained only for GPL attribution and historical reference.
